# URLスライドショーアプリケーション

URLのリストを順番に自動表示するWebアプリケーションです。

## 機能

- **URL管理**: URLの追加、削除、順序変更
- **自動表示**: 設定した時間ごとにURLを順番に表示
- **ループ再生**: 最後のURLまで表示したら最初に戻る機能
- **フルスクリーン表示**: フルスクリーンモードで表示
- **データ管理**: URLリストのエクスポート/インポート（JSON形式）
- **自動保存**: ブラウザのローカルストレージに自動保存

## 使用方法

### 1. アプリケーションの起動

`index.html`をブラウザで開いてください。

### 2. URLの追加

1. 「URLを追加」欄にURLを入力
2. 「追加」ボタンをクリック、またはEnterキーを押す
3. URLがリストに追加されます

### 3. 表示設定

- **表示時間**: 各URLを表示する時間（秒単位）を設定
- **ループ再生**: チェックを入れると、最後のURLまで表示したら最初に戻ります

### 4. URLリストの管理

- **順序変更**: ↑/↓ボタンでURLの順序を変更
- **削除**: 「削除」ボタンでURLを削除

### 5. スライドショーの開始

1. 「開始」ボタンをクリック
2. URLが順番に表示されます
3. 「停止」ボタンで停止できます

### 6. フルスクリーン表示

「フルスクリーン」ボタンをクリックすると、フルスクリーンモードで表示されます。
ESCキーでフルスクリーンを解除できます。

### 7. データのエクスポート/インポート

- **エクスポート**: 「エクスポート」ボタンをクリックすると、現在のURLリストと設定をJSONファイルとしてダウンロードします
- **インポート**: 「インポート」ボタンをクリックして、以前エクスポートしたJSONファイルを選択すると、URLリストと設定を復元できます

## キーボードショートカット

- **スペースキー**: スライドショーの開始/停止
- **ESCキー**: フルスクリーンの解除

## 技術仕様

- **HTML5**: 構造
- **CSS3**: スタイリング（レスポンシブ対応）
- **Vanilla JavaScript**: ロジック実装
- **LocalStorage API**: データの永続化
- **Fullscreen API**: フルスクリーン表示

## ブラウザ対応

モダンブラウザ（Chrome、Firefox、Safari、Edge）で動作します。

## 注意事項

- 一部のWebサイトは、iframeでの表示を制限している場合があります（X-Frame-Optionsヘッダーなど）
- そのようなサイトは、新しいタブで開くか、直接アクセスしてください
- ローカルストレージのデータは、ブラウザの設定によってクリアされる場合があります

## ライセンス

このプロジェクトは自由に使用・改変できます。
